## âš”ï¸ Game Concept

**Medieval Duel** is a 2D, turn-based strategy game where two players (or player vs. AI) clash on two separate 5x5 grids.

The game combines classic auto-chess mechanics with a faster, more action-oriented duel format:

1.  **Preparation Phase:** Each player strategically places their units (knights, archers, mages, etc.) on their own 5x5 grid.
2.  **Battle Phase:** When preparation ends, the "Battle" begins. Units from both sides automatically fight each other based on predefined rules (e.g., closest target, special abilities).
3.  **Objective:** Win the round by eliminating all opposing units and achieve final victory.
